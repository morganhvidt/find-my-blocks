# NavigationItem

`NavigationItem` is a layout component that is used to show information about
a block. It is heavily used in the Sidebar component. 

## Usage

```ts
import { NavigationItem } from '../components/NavigationItem';
```

<Playground>
{() => {
    const [active, setActive] = useState(true);
    return (
        <>
        <NavigationItem label="Toronto Raptors" postCount={1} active={active} onClick={() => setActive(!active)} />
        <NavigationItem label="Los Angeles Lakers" postCount={1} />
        <NavigationItem label="Miami Heat" postCount={2} />
        <NavigationItem label="Boston Celtics" postCount={3} />
        <NavigationItem label="Brooklyn Nets" postCount={4} />
        </>
    )
}}
</Playground>
